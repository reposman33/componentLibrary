Taksu is een stichting die ten doel heeft om educatieve projecten en onderwijs in Bali, Indonesie te steunen. Ik maakte deze website, waarop de stichting zich presenteert, in mijn vrije tijd om wat meer ervaring op te doen met website design en Javascript programmeren. Voor het beheren van de content wordt hetzelfde CMS gebruikt als voor eenschoolintogo.nl. Deze website wordt echter gehost op een Coldfusion platform en haalt daarom data op uit de MySQL database d.m.v. Coldfusion ipv Ajax calls. Een school in Togo is een Amsterdamse stichting die ten doel heeft om goed basis onderwijs bereikbaar te maken voor alle kinderen in Togo. Een trouwe schare fans doneert en middels deze website wordt zij en het algemene publiek op de hoogte gehouden van de gang van zaken.
De website wordt bijgehouden via een CMS. De publieke website bestaat uit statische html en vraagt de content op aan het CMS via JQuery Ajax aanroepen zodat ze zonder server-side scripting toch dynamisch is. (2011): eenschoolintogo.nl.
