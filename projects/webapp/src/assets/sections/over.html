Hallo bezoeker! Om mijn AngularJS activiteiten wat zichtbaarder te maken heb ik een tooltje ontwikkeld in Angular: de ChangeYourSite tool. Bestaande uit 2 ng-apps die ieder een directive bevatten: eentje waarmee je een element kunt verslepen (versleep dit venster maar) en eentje waarmee je de CSS rules van een site kan aanpassen (klik op de button). Je kan zelfs de oude website weer aanzetten!
